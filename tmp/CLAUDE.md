# macnotesapp - CLAUDE.md

## Project Overview

`macnotesapp` is a CLI tool and Python library for interacting with Apple Notes.app on macOS.

- Entry point: `notes` CLI command (`macnotesapp/__main__.py`)
- Python package: `macnotesapp/`
- Current version: `0.8.2` (managed by `bump2version`)
- Python requirement: `>=3.10, <3.14`

## Development Setup

```bash
# Install uv (if not installed)
curl -LsSf https://astral.sh/uv/install.sh | sh

# Create virtualenv and install all dependencies (incl. dev)
uv sync

# Run CLI directly
uv run notes --help

# Or activate virtualenv manually
source .venv/bin/activate
notes --help
```

## Building

### Python Package (wheel + sdist)

```bash
uv build
# output: dist/macnotesapp-<version>.tar.gz and .whl
```

### Standalone CLI Executable (via PyInstaller)

```bash
uv run pyinstaller macnotesapp.spec
# output: dist/notes (standalone binary, no Python needed)
```

### Full Release Build (docs + package + executable + zip + homebrew sha256 update)

```bash
./build.sh
```

`build.sh` sequence:
1. Clean `dist/` and `build/`
2. Update `README.md` via `cog` (auto-generates CLI help text)
3. Build docs via `mkdocs build`
4. Build Python package via `uv build`
5. Build standalone binary via `pyinstaller macnotesapp.spec`
6. Zip the binary as `dist/notes.zip`
7. Update SHA256 in `HomebrewFormula/macnotesapp.rb`

## Installing

### Recommended (via uv tool)

```bash
uv tool install --python 3.13 macnotesapp

# Upgrade
uv tool upgrade macnotesapp

# Run without installing
uvx --python 3.13 macnotesapp
```

### Homebrew (Apple Silicon only)

```bash
brew tap RhetTbull/macnotesapp https://github.com/RhetTbull/macnotesapp
brew update
brew install macnotesapp
```

The Homebrew formula downloads the prebuilt `notes.zip` from GitHub Releases.

## Version Management

`bump2version` synchronizes version across three files:
- `macnotesapp/_version.py`
- `pyproject.toml`
- `HomebrewFormula/macnotesapp.rb`

```bash
uv run bump2version patch --verbose   # e.g. 0.8.2 в†’ 0.8.3
uv run bump2version minor --verbose   # e.g. 0.8.2 в†’ 0.9.0
uv run bump2version major --verbose   # e.g. 0.8.2 в†’ 1.0.0
```
